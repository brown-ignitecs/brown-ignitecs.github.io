Here are reminders of what we talked about in class:

  • HTML creates websites by using commands in brackets <> to understand how to make a website
  • All commands need to have a start <example command> and an end </example command>
  • Everything in between the start and end will have the effect.
    • Example: the command for bold is <b> so
          "Here is a <b>bold</b> word!"
      looks like
          "Here is a bold word!"
      Only the word in between the start <b> and end </b> is bold

Something going wrong?

  • If things seem to be half working: Check and make sure every start has an end
  • If new changes aren't showing up: Make sure to rerun the page!
  • Your enters aren't showing up: Type <br/> instead of pressing the enter
