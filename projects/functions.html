Here are reminders of what we talked about in class:

  • Functions take in information and respond with information
    • For example, in the add function we take in 2 numbers and respond with what the two numbers added together would be
  • The information it takes in is called parameters
    • In the add function the num1, num2 are the parameters in the line function add(num1, num2){}
  • To respond with information, you write return information;
    • For example, if we wanted to respond, "Hi", we would write return "Hi";
